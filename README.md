##### ðŸ“¦ Installation and Running

This project requires Node.js to run.

1. Clone the repository:

   ```bash
   git clone <repo-url>
   cd jazz-jackrabbit
   ```

2. Install the required packages:

   ```bash
   npm install
   ```

3. Start in development mode:

   ```bash
   npm run dev
   ```

4. To access it on a local network:

   ```bash
   npm run host
   ```

5. Build for production:

   ```bash
   npm run build
   ```

6. Preview the build:
   ```bash
   npm run preview
   ```
